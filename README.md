## О проекте

## Быстрый запуск

Сделайте клон проекта к себе на компьютер:

```
git clone https://github.com/SidorovWeb/Webpack-5-Pug-3-Sass-JavaScript-React.git new-project-name
```

```
cd new-project-name
npm i
```

Запустите проект:

```
npm run serve
```

Собрать проект:

```
npm run build
```
